Chrome extension to search highlighted text with Jellyneo in the context menu.

![Example Gif](media/example.gif)

# How to use

1. Clone/download this project to a location of your choice. If download, unzip
2. Open "Manage Extensions" in your Chromium-based browser (typically at [chrome://extensions](chrome://extensions), [brave://extensions](brave://extensions), [edge://extensions](edge://extensions) etc.)
3. Find the "Developer mode" toggle and switch it on
4. Click "Load unpacked"
5. Select the project folder
6. Voila!
